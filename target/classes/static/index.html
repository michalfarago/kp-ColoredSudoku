About Game

Welcome to my color Sudoku. Place each of the colours in the grid so that every row, every column, and each of the nine square contains only one occurence of all the colours. All puzzles have only one solution. If you feel you can't go, you can click the 'HINT' button, which will tell you where should you put certain color.
